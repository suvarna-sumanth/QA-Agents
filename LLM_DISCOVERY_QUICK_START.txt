HOW IT WORKS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1: Discover URLs (Fast)
├─ Tries: Sitemap XML
├─ Then: RSS/Atom feed
└─ Or: Homepage crawl with Chrome

Step 2: LLM Intelligence (Smart) ← NEW!
├─ Analyzes each URL
├─ Identifies real articles
├─ Filters out: nav pages, tags, about, etc.
└─ Returns: Top N articles, most recent first

Step 3: Fallback (Reliable)
└─ If LLM fails: Uses heuristic URL patterns
